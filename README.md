# ChatGPT
### ChatGPT bot for telegram

### How to start
All you need is:
1) install dependencies (python libraries):  
• aiogram  
• openai  
• logging  
just use pip install library_name (or pip3 install in unix systems);  
2) create your own bot with https://t.me/BotFather;  
3) create openai account, create api key;
